# 🌍 ClimaSim — Modelo Iterativo de Cambio Climático

Proyecto escolar CECyTEA. Simulador de cambio climático con modelo de diferencias finitas.

## Estructura
- `main.js` — Proceso principal de Electron
- `index.html` — UI completa (HTML/CSS/JS)

## Ejecutar localmente
```bash
npm install
npm start
```

## Generar .exe
```bash
npm install
npm run build
# El .exe queda en /dist
```
